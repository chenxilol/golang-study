# Mutex 与 RWMutex 作用

# Mutex 与 RWMutex 的应用场景
1. 协程安全问题
2. 数据竞争问题

2. 应合理使用锁机制， 不要滥用

# sync.Map 
1. 一个线程安全集合，内部通过原子访问和锁机制实现结合的线程安全

# sync.Map 的应用场景
1. 适合读多写少的应用场景
2. 在key值以存在的情况下，可以无修改其value，比普通 map + 锁性能更好
